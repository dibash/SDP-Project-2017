Detailed Description

A class representing a Non-deterministic Finite Automata.

Simulates an NFA, constructed using Thompson's construction from a regular expression of the form: 

    {valid-sym} := any ASCII symbol with code from 33 to 126 (printable characters)
    {base-sym}  := any {valid-sym} except ( or ) or | or . or * or \
    {spec-sym}  := \s or \d or \a or \e or \\
    {symbol}    := {base-sym} or {spec-sym}
    {sym-seq}   := {symbol} or {symbol}
    {reg-expr}  := {sym-seq}               or
                   ({reg-expr}.{reg-expr}) or
                   ({reg-expr}|{reg-expr}) or
                   ({reg-expr})*
Note: '?' added as a wildcard symbol (matches any character)


The states are named 0 to N, where 0 is the begin state and N is the final state. N is also the size of the regular expression after parsing. That way each symbol from the parsed expression corresponds to a state in the NFA. For this implementation the index of the symbol in the regular expression is also the name of the corresponding state.
